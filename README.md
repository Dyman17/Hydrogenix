# Hydrogenix - Экологичный гидрогель для оптимизации полива растений

Hydrogenix - это инновационный экологичный гидрогель, разработанный для оптимизации процесса полива растений. Продукт изготовлен из натуральных компонентов и помогает удерживать влагу в почве, обеспечивая растения постепенным поступлением воды.

## Особенности

- ✅ Экологичный состав на основе природных компонентов
- ✅ Удержание влаги в почве до 30-45 дней
- ✅ Подходит для домашних растений, сельского хозяйства и деревьев
- ✅ 100% биоразлагаемость
- ✅ Экономия воды и времени на полив

## Технологии

- **React** - современный фреймворк для создания пользовательских интерфейсов
- **TypeScript** - строгая типизация для надежного кода
- **Tailwind CSS** - utility-first CSS фреймворк для стилизации
- **Vite** - быстрая сборка и разработка
- **Shadcn/ui** - переиспользуемые компоненты с возможностью настройки
- **Lucide React** - библиотека иконок

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Dyman17/hydrogenix-grow-smarter
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект в режиме разработки:
```bash
npm run dev
```

## Сборка

Для сборки проекта в продакшен:
```bash
npm run build
```

## Структура проекта

```
├── public/                 # Статические файлы
├── src/
│   ├── components/         # Переиспользуемые компоненты
│   │   ├── home/          # Компоненты главной страницы
│   │   ├── layout/        # Компоненты макета (шапка, футер)
│   │   └── ui/            # UI компоненты из shadcn
│   ├── contexts/          # React контексты
│   ├── data/              # Данные приложения
│   ├── hooks/             # Пользовательские хуки
│   ├── lib/               # Вспомогательные функции
│   ├── pages/             # Страницы приложения
│   ├── App.tsx            # Главный компонент приложения
│   └── main.tsx           # Точка входа в приложение
├── .env.example           # Пример переменных окружения
├── package.json           # Зависимости и скрипты
└── vite.config.ts         # Конфигурация сборки
```
